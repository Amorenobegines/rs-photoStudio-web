RS PhotoStudio – Página Web
Proyecto web desarrollado para el estudio fotográfico RS PhotoStudio, con el objetivo de crear una plataforma moderna, 
visual y fácil de gestionar donde mostrar servicios, sesiones y contenido del blog.

Características principales
Estructura desarrollada con Node.js y Express.

Vistas dinámicas creadas con EJS.

Diseño moderno y responsive gracias a TailwindCSS.

Sistema de blog basado en objetos JavaScript para gestionar publicaciones.

Organización clara de rutas, vistas y archivos estáticos.

Tecnologías utilizadas
Node.js

Express

EJS

TailwindCSS

JavaScript (ES6+)

Estructura del proyecto
/routes – Rutas principales de la web.

/views – Plantillas EJS organizadas por secciones.

/public – Archivos estáticos (CSS, imágenes, scripts).

app.js – Configuración del servidor y middlewares.

Cómo ejecutar el proyecto

Sobre el proyecto
Este sitio web está diseñado para mostrar el trabajo fotográfico de RS PhotoStudio, incluyendo sesiones de familia, maternidad, newborn, comuniones y contenido de blog. El objetivo es ofrecer una experiencia visual cuidada y una navegación sencilla para los usuarios.
